(`sales/index.html`) became the single canonical sales page.

These are **snapshots, not live pages** — nothing links to them. Their relative
asset/CSS paths point at the original `sales/` location (`ai-forward.css`,
`../assets/…`); if you revive one, move it back up to `sales/` or fix the paths.
